# Readme

## Getting started

The project was built using the following versions

- `truffle: 5.1.59`
- `solidity: 0.7.4`
- `ganache (CLI): 6.12.1`
- `node: 14.4.0`
- `yarn: 1.22.4`
- `TypeScript: 4.1.3`
- `MetaMask: 9.0.5`

## Mint tokens

To mint tokens, I created small application that can be started with `yarn start`. For the token owner it the possible to mint new tokens here.

The contract can be found here: https://rinkeby.etherscan.io/address/0xdabc0585ac05418acec705dfa93d032070bdc40e

The minted token are available at https://testnets.opensea.io/collection/udacitycapstone on OpenSea.
